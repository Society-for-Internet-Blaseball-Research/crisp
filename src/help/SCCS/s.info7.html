Future Directions

The CRiSP model is programmed with a robust and flexible structure that allows for easy future accommodation of new theory with a minimum of reprogramming effort.

The following activities are planned for the near future:

  • improving the graphical interface to provide additional on-line help and output capabilities
  • improving some key functional relationships in the computation engine
  • implementing parameter estimation capabilities
  • developing an interactive tool for viewing codedwire-tag data for modeled stocks
  • linking CRiSP.1 and CRiSP Harvest
